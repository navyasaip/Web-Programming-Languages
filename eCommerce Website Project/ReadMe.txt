A responsive E-commerce site utilizing Service-Oriented MVC architecture, RESTful Web Services, JSP, Ajax, Bootstrap
other web programming languages like HTML, CSS and jQuery.

User login, search functionality, shopping cart, add/update reviews functionalities.

Utilized Hibernate (ORM Framework) in establishing database connection and caching such as MemCached for better performance. 

Mounted client and server on two different ports (8080/8443 and 8081/9443), encrypted (TLS/SSL) 
gzip compressed the client-server communication.
